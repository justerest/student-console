▸ readln(message, defaultValue?)

Reads the line entered by user. Returns a string.

Считывает строку, введённую пользователем. Возвращает строку.

readln(message: string, defaultValue: string?): Promise<string>
Parameters
message (string)

▸ readlnInt(message, defaultValue?)

Reads the line entered by user. Returns a number.

Считывает строку, введённую пользователем. Возвращает целое число.

readlnInt(message: string, defaultValue: number?): Promise<number>
Parameters
message (string)
defaultValue (number?) The value to be assigned in dev mode. Значение, которое будет присваиваться в dev-режиме.
Returns
Promise<number>:
Throws
  • any: Will throw an error and exit if can't convert the line into number. Вызовет ошибку и завершит программу, если не удастся конвертировать строку в число.
Example
const yearsOld = await stdio.readlnInt('Сколько Вам лет? ', 15);
▸ write(message)

Print a line on the screen. Next output will start from the current line.

Вывод строки на экран. Следующий вывод начнётся с текущей строки.

write(message: (string | number))
Parameters
message ((string | number))
Example
const result = 15;
write('Ответ: ');
writeln(result);
